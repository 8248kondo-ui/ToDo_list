# 📅 優先度＆カレンダー付きToDoアプリ

PythonとStreamlitで作成した、シンプルで使いやすいToDo管理アプリです。
タスクに優先度（高・中・低）と期限を設定でき、データはCSVファイルに自動保存されます。

## 🌟 主な機能
- **タスク管理**: 追加、表示、個別削除、一括削除。
- **優先度設定**: 優先度に応じて色分け表示（赤・オレンジ・青）。
- **期限管理**: カレンダーから期限を選択。期限が近いタスクはサイドバーで警告。
- **自動保存**: アプリを閉じてもデータが消えないCSV保存機能。

## 🚀 使い方
1. 以下のURLにアクセスします。
   (ここにあなたのStreamlit CloudのURLを貼る)
2. 「タスクを追加する」から、タスク名、期限、優先度を選んで追加ボタンを押します。
3. 完了したタスクはチェックボックスで完了リストへ移動できます。

## 🛠️ 使用技術
- **Python 3.13**
- **Streamlit** (Webフレームワーク)
- **Pandas** (データ処理)
